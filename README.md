# Diabetes Prediction using Machine Learning

## Overview
This project focuses on predicting the likelihood of diabetes using machine learning techniques applied to the TUDA dataset. The goal is to analyze health-related features and build a model that can classify whether a patient is likely to have diabetes.

## Features
- Data preprocessing and cleaning
- Exploratory data analysis
- Machine learning model training
- Model evaluation and performance analysis

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Dataset
The dataset used in this project is the TUDA diabetes dataset, which contains medical and demographic features used for prediction.

## Project Structure
- `diabetes_prediction.ipynb` → main notebook with code and analysis  
- `diabetes_dataset.csv` → dataset used for training  
- `report.pdf` → detailed project report  

## Results
The model was trained and evaluated using standard machine learning techniques to predict diabetes based on input features. Performance was analyzed using appropriate evaluation metrics.

## Why I Built This
This project was developed as part of my Artificial Intelligence coursework to apply machine learning concepts to a real-world healthcare problem. It helped me understand data preprocessing, model building, and evaluation in practice.
